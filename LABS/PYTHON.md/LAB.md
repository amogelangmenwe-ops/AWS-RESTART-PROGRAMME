Using AWS Cloud9 to Work with Python
-
To start working with Python, I used AWS Cloud9, which is a cloud-based development environment provided by AWS. After opening Cloud9 from the AWS Services menu, I accessed the existing environment and launched the terminal. From there, I was able to run Python directly in the cloud without installing anything on my local system.

Using Cloud9 made it easy to open Python, write code, and execute programs in real time. The environment provided a code editor, terminal, and file management tools all in one place. This helped me focus more on learning Python concepts rather than worrying about setup or configuration issues.

Working in Cloud9 also showed me how Python can be used in a cloud environment, which is useful for automation and cloud-based tasks. Seeing my code run successfully in AWS helped me better understand how Python is applied in real-world cloud scenarios.

Challenges Faced
-
While learning Python in the AWS Cloud9 environment, I faced several challenges. One of the main difficulties was understanding Python syntax and indentation, as even small mistakes could cause errors. I also found it challenging to debug errors, especially when the code did not run as expected at first.

Another challenge was writing correct logic for loops and conditional statements, which required careful thinking and practice. At times, remembering the correct structure of functions and how to use them properly was also difficult. Additionally, working in a cloud-based environment was new to me, so getting familiar with Cloud9 tools and the terminal took some time.

Despite these challenges, continuous practice and experimentation helped me improve and become more confident in using Python.

Solutions and How I Overcame Them
-
To overcome the challenges I faced while learning Python in AWS Cloud9, I took several steps:

Practiced coding regularly to get familiar with Python syntax and indentation.

Carefully read error messages to understand what went wrong and how to fix it.

Broke problems into smaller steps to make complex logic easier to handle.

Re-ran code multiple times to test solutions and improve accuracy.

Referred to examples, tutorials, and course instructions whenever I was unsure.

Experimented in the Cloud9 environment to get comfortable with running Python in the cloud.

These strategies helped me improve my coding skills, solve problems independently, and gain confidence in using Python in a cloud-based environment.
